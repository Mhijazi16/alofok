   - **Collapsed state:** Glass card showing receipt icon, "Expenses" title, and today's total amount (sum of expenses). Tap anywhere to expand. Use `Collapsible` from Radix or simple `useState<boolean>` toggle. Show a chevron indicator for expand/collapse.
   - **Expanded state:** Shows list of expense rows for the date + an "Add Expense" button at the bottom.
   - **Each expense row:** Shows category icon (colored), category name (localized via `t("expense.category.{key}")`), amount in ILS, date, and a status badge (pending=yellow, confirmed=green, flagged=red). If not admin and status is "pending", show a delete (Trash2) icon button.
   - **Add button:** Opens a Dialog with:
     - Category grid: 2x3 or 3x3 grid of category buttons, each showing the icon on its colored background + localized name below. Selected category has a ring/border highlight.

     - Date picker: DatePicker component, defaults to the card's `date` prop
     - Notes: Optional textarea
     - Submit button: Calls `salesApi.createExpense()`, invalidates `["my-expenses"]` and `["daily-ledger"]` query keys on success, shows toast, closes dialog
   - **Delete:** Calls `salesApi.deleteExpense(id)`, invalidates same query keys, shows toast
   - **Data fetching:** Use `useQuery({ queryKey: ["my-expenses", date], queryFn: () => salesApi.getMyExpenses(date) })`. Only fetches when the card's date changes.
   - **Empty state:** When no expenses, show a subtle message like "No expenses for this day" in the expanded section.
   - **Loading:** Skeleton placeholders while loading.
