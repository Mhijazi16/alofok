**Implementation:**

   - Camera input: `accept="image/*" capture="environment"` — triggers rear camera
   - Gallery input: `accept="image/*"` (no capture) — triggers file picker/gallery
2. Two icon buttons side by side (Camera icon + ImageIcon from lucide-react), each clicking its respective hidden input
3. `handleFile(file: File | undefined)` handler:

   - Create preview URL via `URL.createObjectURL(compressedBlob)`
   - Call `onCapture(compressedBlob, previewUrl)`
4. When `imagePreviewUrl` is set, show:
   - Large preview image (~40-50% container width, `rounded-lg object-cover border border-border`)
   - Retake button (RefreshCw icon) — clears current and re-opens camera input
   - Remove button (X icon) — calls `onRemove()`
   - If `onScanCheck` prop provided, show "Scan Check" button (ScanLine icon) below preview
   - If `isScanning` is true, show spinner overlay on the preview image (absolute positioned, `bg-black/50 rounded-lg` with Spinner component)
5. When no image, show just the camera + gallery buttons with a subtle label
6. Use `useTranslation` for all text strings
7. Reset input values after file selection (set `input.value = ""`) so the same file can be re-selected
