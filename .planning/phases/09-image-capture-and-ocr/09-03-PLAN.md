Create `frontend/src/hooks/useOcr.ts`:

1. **Export types:**
   ```typescript
   export type OcrConfidenceLevel = "high" | "medium" | "low";
   export interface OcrFieldResult {
     value: string;
     confidence: OcrConfidenceLevel;
   }
   export interface OcrResult {
     bankNumber?: OcrFieldResult;
     branchNumber?: OcrFieldResult;
     accountNumber?: OcrFieldResult;
     amount?: OcrFieldResult;
     holderName?: OcrFieldResult;
   }
   ```

2. **Confidence mapping function:**
   - `toConfidenceLevel(score: number)`: >= 70 -> "high", >= 40 -> "medium", < 40 -> "low"

3. **`useOcr()` hook:**
   - `workerRef = useRef<Worker | null>(null)` for persistent worker across scans
   - `isScanning` state (boolean)
   - `error` state (string | null)
   - `scan(imageBlob: Blob): Promise<OcrResult>` function:
     - Sets isScanning true, clears error
     - Lazily creates worker if not exists: `createWorker(["eng", "ara"], 1, { cacheMethod: "write" })`
     - Calls `worker.recognize(imageBlob)`
     - Passes `data` to `extractCheckFields()` function
     - On error: sets error message, returns empty `{}`
     - Finally: sets isScanning false
   - Cleanup effect: `useEffect(() => { return () => { workerRef.current?.terminate(); }; }, [])`
   - Returns `{ scan, isScanning, error }`

4. **`extractCheckFields(data: Tesseract.Page): OcrResult`** function:
   - Extract all recognized text from `data.text`
   - Extract words with confidence from `data.words`
   - **Amount detection:** regex for decimal number pattern `\b(\d{1,6}[.,]\d{2})\b` — map to amount field with avg confidence of matching words
   - **Numeric groups:** find all `\b(\d{2,10})\b` sequences for bank/branch/account numbers
   - **MICR line heuristic:** Look for numeric sequences near the bottom of the image (by y-coordinate in `word.bbox`). Bottom-third words with numeric content are likely MICR: assign first group as branch, second as account. If only found generically (no bbox filtering), assign in order.
   - **Holder name:** Look for non-numeric text sequences (2+ words) in the upper portion of the image. Use avg confidence of those words. Skip if primarily Arabic characters detected and only eng worker loaded.
   - **Bank number:** Look for short numeric sequence (2-4 digits) that is NOT the amount, branch, or account — typically the bank code
   - All extraction is best-effort with confidence scores communicating reliability
   - Log extracted fields at debug level for development: `console.debug("[OCR] extracted:", result)`

5. **Export `confidenceBorderClass` utility:**
   ```typescript
   export function confidenceBorderClass(confidence: OcrConfidenceLevel | null): string {
     if (confidence === "high") return "ring-2 ring-green-500";
     if (confidence === "medium") return "ring-2 ring-yellow-500";
     if (confidence === "low") return "ring-2 ring-red-500";
     return "";
   }
   ```
