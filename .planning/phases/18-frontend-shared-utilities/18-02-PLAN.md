Create `frontend/src/hooks/useCart.ts` that encapsulates the cart state pattern duplicated in Sales/index.tsx (lines 462-526), Customer/index.tsx (lines 242-306), and Admin/index.tsx (lines 89-124).

The hook signature:
```typescript
interface UseCartOptions {
  /** localStorage key for persistence. Omit or pass null to skip persistence. */
  storageKey?: string | null;
}

interface UseCartReturn {
  cart: Map<string, CartItem>;
  addToCart: (product: Product, qty?: number, selectedOptions?: SelectedOption[]) => void;
  updateCartQty: (key: string, qty: number) => void;
  removeFromCart: (key: string) => void;
  clearCart: () => void;
  cartTotal: number;
}

export function useCart(options?: UseCartOptions): UseCartReturn;
```

Implementation details:
- Accept optional `storageKey` parameter. If provided, initialize state from localStorage and persist via useEffect (matching existing Sales "alofok-cart" and Customer "alofok-customer-cart" patterns). If omitted/null, start with empty Map and skip persistence (Admin pattern).
- `addToCart`: Uses `cartKey(product.id, selectedOptions)` to compute map key. If key exists, increment quantity. Otherwise, insert new entry.
- `updateCartQty`: If qty <= 0, delete key. Otherwise update existing entry's quantity.
- `removeFromCart`: Delete key from map.
- `clearCart`: Reset to empty Map.
- `cartTotal`: useMemo over cart values: `sum of ((discounted_price ?? price) + optionsPrice(selectedOptions)) * quantity`.
- All mutators wrapped in useCallback.
- Import CartItem, Product, SelectedOption from `@/services/salesApi` and cartKey, optionsPrice from `@/lib/cart`.
