  <name>Task 1: Add Custom date range tab to Sales StatementView</name>
  <files>frontend/src/components/Sales/StatementView.tsx</files>
  <action>
Modify the Sales StatementView to add a 5th "Custom" tab with date range picker:

1. **Extend FilterPreset type:** Change to `"zero" | "week" | "month" | "year" | "custom"`.

2. **Add state:** `const [customRange, setCustomRange] = useState<DateRange | undefined>();`
   Import `DateRange` from `react-day-picker` and `DatePicker` from `@/components/ui/date-picker`.

3. **Update queryParams useMemo:** Add a `case "custom":` that returns `{ start_date: toLocalDateStr(customRange.from), end_date: toLocalDateStr(customRange.to) }` when both from and to are defined. If either is undefined, return `{ since_zero_balance: true }` as a safe fallback (prevents empty query while user is still selecting).

4. **Update queryKey:** Change from `["statement", customer.id, preset]` to `["statement", customer.id, preset, customRange?.from?.toISOString(), customRange?.to?.toISOString()]`. This ensures React Query refetches when custom range changes. Per research pitfall #3 -- if custom range is not in the key, stale data appears.

5. **Add 5th TabsTrigger:** After the "year" trigger, add `<TabsTrigger value="custom">{t("statement.customRange")}</TabsTrigger>`. The locale key "statement.customRange" already exists in ar.json as "فترة مخصصة".

6. **Show DatePicker conditionally:** Below the Tabs component, add:
   ```tsx
   {preset === "custom" && (
     <DatePicker
       mode="range"
       value={customRange}
       onChange={setCustomRange}
       placeholder={t("statement.selectDateRange")}
     />
   )}
   ```
   The DatePicker uses Radix Popover.Portal so it renders outside the parent container (no overflow clipping issues per research pitfall #4).

7. **Preserve custom range:** The customRange state is already component-level -- switching to another tab and back to "custom" restores the range automatically (React state persists across tab switches within the same component mount).
