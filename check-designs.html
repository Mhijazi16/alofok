Design E — Watermark Variants

Same base layout (gradient accent, pill amount). Comparing watermark styles. Pick your favorite.

E1 — Floating Circles

Variant

Soft overlapping circles of varying sizes scattered across the body. Like bubbles or bokeh — organic and airy.

Feel: Light, playful, organic. The circles drift like soft watermarks behind a real check.

E2 — Diamond Lattice

Variant

Repeating rotated squares form a diamond pattern. Classic security watermark motif — like the pattern inside real banknotes.

Feel: Structured, professional, bank-like. The repeating pattern feels like a security feature.

E3 — Swooping Curves

Variant

Flowing bezier curves sweep across the check like ribbon trails. Elegant, almost calligraphic. Reminiscent of currency engraving patterns.

Feel: Elegant, flowing, currency-inspired. The thick transparent strokes + thin visible edges create depth.

E4 — Geometric Scatter

Variant

Mix of shapes — hexagons, triangles, rounded squares — scattered at various rotations. Modern, playful, like a branded pattern you'd see on fintech apps.

Feel: Fintech-modern. Mixed shapes add visual interest without competing with text. Feels branded and intentional.

E5 — Concentric Rings

Variant

Two large ring clusters — one behind the bank zone, one near the amount. Like ripples in water. Minimal but with depth.

Feel: Calm, centered, elegant. Two focal points create visual balance. Feels like a watermark seal.
